# applications_user_certificate_manager

#### Description
用户证书管理应用是OpenHarmony中预置的系统应用，主要的功能包含：
1、系统预置CA证书的查看功能；
2、用户CA证书的安装、查看、启用禁用与卸载功能；
3、用户证书凭据的安装、查看、授权管理与卸载
4、系统证书凭据（Wlan/vpn证书凭据）的安装、查看、卸载功能
